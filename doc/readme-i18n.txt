
i18n File skeleton

* usr/locale: Global, locale folder name must be in lower case
  * /en
    * default.mo: General
    * navigation.mo: Navigation and menu
    * ...
  * /zh-cn
    * ...
* usr/module/demo: Module, locale folder name must be in lower case
  * /en
    * /mail: Mail templates
      * mail-template.phtml
    * default.mo
    * navigation.mo
    * admin.mo: Admin area
    * config.mo: Config edit
    * feed.mo: Feed
    * ...
  * /zh-cn
    * ...


i18n File skeleton

* usr/locale: Global, locale folder name must be in lower case
  * /en
    * main.csv: General
    * navigation.csv: Navigation and menu
    * ...
  * /zh-cn
    * ...
* usr/module/demo: Module, locale folder name must be in lower case
  * /en
    * /mail: Mail templates
      * mail-template.phtml
    * main.csv
    * navigation.csv
    * admin.csv: Admin area
    * config.csv: Config edit
    * feed.csv: Feed
    * ...
  * /zh-cn
    * ...
